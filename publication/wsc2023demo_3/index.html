PyCDSL: A Programmatic Interface to Cologne Digital Sanskrit Dictionaries

Hrishikesh Terdalkar, Arnab Bhattacharya
January, 2023
Cite Project PDF PyPI Documentation Code

Abstract

PyCDSL is a Python library that provides programmer friendly interface to Cologne Digital Sanskrit Dictionaries (CDSD). The library serves as a corpus management tool to download, update and access dictionaries from CDSD. The tool provides a command line interface for ease of search and a programmable interface for using CDSD in computational linguistic projects written in Python 3.
