PyCDSL
PyCDSL is a python interface to Cologne Digital Sanskrit Lexicon (CDSL). It provides an unified method to download, update and programmatically access all the dictionaries available at CDSL. It also features a console command and a REPL interface to make dictionary search easy for non-programmers. PyCDSL has extensive support for transliteration using indic-transliteration module.
Hrishikesh Terdalkar
